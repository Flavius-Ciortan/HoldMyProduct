# WordPress.org Assets

This folder contains assets for the WordPress.org plugin directory.

## Required Assets

### Icons
- `icon-128x128.png` - Small icon (128x128px)
- `icon-256x256.png` - Large icon (256x256px)

### Banners
- `banner-772x250.png` - Standard banner (772x250px)
- `banner-1544x500.png` - Retina banner (1544x500px)

### Screenshots

## Design Guidelines

### Icon
- Use the plugin's primary color (#008B8B - Dark Cyan)
- Should be recognizable at small sizes
- Suggested: Shopping cart with a clock or "hold" symbol
- PNG format with transparent background

### Banner
- Feature product reservation concept
- Use brand colors (#008B8B primary, #20B2AA secondary)
- Include plugin name "HoldThisProduct"
- Show key benefit: "Reserve Products, Guarantee Stock"

### Screenshots
- Use actual plugin screenshots
- Capture at standard desktop resolution (1920x1080 recommended)
- Show real functionality, not mockups
- Include clear examples of features in action

## Upload Instructions

When submitting to WordPress.org:
1. These assets are NOT included in the plugin ZIP
2. Upload to SVN assets folder separately
3. WordPress.org will automatically use them in the directory listing

## Notes

The `.wordpress-org` folder is automatically excluded from the plugin ZIP by WordPress.org deployment tools.
